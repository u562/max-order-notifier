== Description ==

**MAX Order Notifier** — это мощный плагин для WooCommerce, который отправляет подробные уведомления о заказах в мессенджер MAX. Теперь вы можете получать всю информацию о заказах прямо в чат, не заходя в админпанель.

= 🌟 Основные возможности =

* ✅ **Автоматическая отправка** — уведомления приходят сразу после оформления заказа
* ✅ **Изменение статусов** — отслеживайте все изменения статусов заказов
* ✅ **Полная информация** — в сообщение включаются все товары, способ доставки, данные клиента
* ✅ **Гибкие настройки** — выбирайте какие статусы отслеживать
* ✅ **Тестовый режим** — проверьте работу плагина одной кнопкой
* ✅ **Подробное логирование** — все события записываются в лог для отладки
* ✅ **Безопасность** — токен бота и chat_id хранятся в зашифрованном виде

= 📦 Что входит в уведомление =

* Номер и дата заказа
* Общая сумма и способ оплаты
* Способ доставки
* Полный список товаров с количеством и ценами
* Данные клиента (ФИО, телефон, email)
* Адрес доставки
* Комментарий к заказу (если есть)
* Прямая ссылка на заказ в админке
