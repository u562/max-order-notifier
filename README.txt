
= 🚀 Почему MAX Order Notifier? =

* **Мгновенное оповещение** — не нужно постоянно проверять почту или админку
* **Удобный формат** — вся информация структурирована и легко читается
* **Экономия времени** — быстрая обработка заказов прямо из мессенджера
* **Надежность** — плагин использует официальное API MAX
* **Поддержка** — регулярные обновления и техническая поддержка

== Installation ==

= Минимальные требования =

* WordPress 5.0 или выше
* WooCommerce 4.0 или выше
* PHP 7.2 или выше
* Аккаунт в мессенджере MAX

= Установка через админ-панель =

1. Загрузите архив плагина
2. Перейдите в **Плагины → Добавить новый → Загрузить плагин**
3. Выберите архив и нажмите "Установить"
4. После установки нажмите "Активировать"

= Установка вручную =

1. Распакуйте архив с плагином
2. Скопируйте папку `max-order-notifier` в `/wp-content/plugins/`
3. Перейдите в **Плагины** и активируйте **MAX Order Notifier**

= Настройка =

1. Перейдите в **WooCommerce → Настройки → Интеграции → MAX Messenger Notifier**
2. Введите **Bot Token** (как получить — см. ниже)
3. Введите **Chat ID** (ID чата или канала)
4. Выберите статусы заказов для отправки уведомлений
5. Нажмите **"Сохранить изменения"**
6. Нажмите **"Отправить тест"** для проверки подключения

= Как получить Bot Token =

1. Установите мессенджер MAX
2. Найдите официального бота для создания ботов (обычно @MasterBot)
3. Отправьте команду `/newbot`
4. Следуйте инструкциям для создания нового бота
5. Скопируйте полученный токен

= Как получить Chat ID =

**Для канала:**
1. Создайте канал в MAX
2. Добавьте туда вашего бота как администратора
3. Chat ID будет отрицательным числом (например: -73583470536035)

**Для личного чата:**
1. Напишите вашему боту любое сообщение
2. Используйте API метод `getUpdates`
3. Найдите ваш `chat_id` в ответе сервера

== Frequently Asked Questions ==

= Вопрос: Какие статусы заказов можно отслеживать? =

**Ответ:** Вы можете выбрать любые статусы из списка: Ожидание оплаты, Обработка, Выполнен, Отменен, Возврат и другие.

= Вопрос: Можно ли отправлять уведомления в несколько чатов? =

**Ответ:** В текущей версии поддерживается отправка только в один чат. Для отправки в несколько чатов используйте групповой чат или канал.

= Вопрос: Безопасно ли хранить токен бота в настройках? =

**Ответ:** Да, токен хранится в базе данных WordPress в зашифрованном виде. Доступ к настройкам имеют только администраторы сайта.

= Вопрос: Что делать, если сообщения не приходят? =

**Ответ:** Проверьте:
1. Правильность токена и Chat ID в настройках
2. Логи плагина в WooCommerce → Статус → Логи
3. Что бот добавлен в чат/канал
4. Что вы выбрали правильные статусы для отправки

= Вопрос: Поддерживает ли плагин отправку изображений? =

**Ответ:** Текущая версия отправляет только текстовые сообщения. Поддержка изображений планируется в следующих обновлениях.

= Вопрос: Как отключить уведомления для определенных статусов? =

**Ответ:** В настройках плагина снимите галочки с ненужных статусов в поле "Статусы заказов для уведомлений".

= Вопрос: Работает ли плагин с HPOS (High-Performance Order Storage)? =

**Ответ:** Да, плагин полностью совместим с новой системой хранения заказов WooCommerce.

= Вопрос: Можно ли изменить текст сообщения? =

**Ответ:** Да, вы можете изменить шаблон сообщения, отредактировав метод `get_full_order_message()` в файле `class-max-order-notifier.php`.
