== Description ==

MAX Order Notifier - это плагин для WooCommerce, который автоматически отправляет уведомления о заказах в мессенджер MAX.

= Возможности =
* Отправка уведомлений о новых заказах
* Отправка уведомлений об изменении статуса заказа
* Настройка для каких статусов отправлять уведомления
* Кастомизация сообщений
* Тестовая отправка для проверки настроек

== Installation ==

1. Загрузите папку `max-order-notifier` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в WooCommerce > Настройки > Интеграции > MAX Messenger Notifier
4. Введите токен вашего MAX бота и ID чата
5. Настройте необходимые статусы заказов для уведомлений

= Как получить токен MAX бота =
1. Установите мессенджер MAX
2. Найдите бота @MasterBot
3. Следуйте инструкциям для создания нового бота
4. Скопируйте полученный токен

== Frequently Asked Questions ==

= Где взять Chat ID? =
После создания бота, напишите ему любое сообщение, затем используйте API метод getUpdates чтобы найти ваш chat_id.

= Поддерживает ли плагин несколько чатов? =
В текущей версии поддерживается отправка только в один чат.
